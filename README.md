## Manual steps

### Clone the repo (inits and updates the submodules)
```bash
git clone --recurse-submodules https://github.com/AKolodeev/ansible.git
```

### Install Homebrew and Ansible
```bash
./ansible-install.sh
```

### Restart the shell

### Run Ansible
```bash
ansible-playbook local.yaml --ask-vault-pass
```
